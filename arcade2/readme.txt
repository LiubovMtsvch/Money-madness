Итоговый проект: Создание игры на Arcade (3.0.0.dev18)

Описание игры: главный персонаж(призрак) собирает биткойны, которые летят сверху.
Главные враги персонажа - черепа, которые летят сверху вместе с валютой. Цель игры заключается в том, 
чтобы собрать как можно больше биткойнов и выжить. Если ваши жизни закончились, то игра 
завершается. Для управления используйте клавиши UP, DOWN, LEFT, RIGHT.

P.S. в терминале вводим pip install arcade==3.0.0.dev18
(новая версия)
